Welcome to my repository! This space serves as a general portfolio of my academic assignments, software engineering labs, and console applications developed in C++. The projects compiled here showcase robust object-oriented concepts, clean code design, and structured problem-solving frameworks.

The codebase focuses on mastering key software development principles such as modular data structuring with classes and structs, persistent data storage via File I/O streams, and efficient resource management. Each project is designed to build foundational skills in executing complex logic, handling input streams dynamically, and maintaining clean code architecture.

How to Compile and Run:

    Clone the repository to your local system.

    Open your terminal and navigate to the project folder.

    Compile using a standard C++ compiler: g++ filename.cpp -o program

    Run the executable (./program on Linux/macOS or program.exe on Windows).
